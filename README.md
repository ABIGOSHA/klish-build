# klish-build

Сборка и установка [klish](https://github.com/ABIGOSHA/klish-config) — гибкого фреймворка для создания CLI-интерфейсов.

## 🛠 Скрипты управления

| Скрипт | Назначение |
|--------|------------|
| `./setup.sh` | Скачивает исходные коды `libxml2`, `faux`, `klish-config` и собирает зависимости для целевой архитектуры. |
| `./build.sh` | Компилирует `klish` с использованием собранных зависимостей и упаковывает результат вместе с конфигурацией. |
| `./install.sh` | Устанавливает собранные файлы в систему (требует root). |

## 📦 Зависимости

- `libxml2` (собирается из исходников)
- `faux` (собирается из исходников)
- Компилятор (gcc), make, wget, git

## 🚀 Быстрый старт

```bash
# Подготовка зависимостей
chmod +x setup.sh
./setup.sh

# Сборка klish
./build.sh

# Установка на Raspberry Pi (запускать на самом устройстве или с sudo)
sudo ./install.sh
